\section*{Selected Projects}

\textbf{Distributed Real-Time Collaboration Platform}

\textit{Real-time distributed state synchronization platform}

\begin{itemize}[leftmargin=*]

\item Engineered a horizontally scalable real-time collaboration platform using raw WebSockets, Redis Pub/Sub, and PostgreSQL, enabling low-latency synchronization across multiple Node.js instances.

\item Designed a hot/cold storage architecture with Redis caching and PostgreSQL persistence, supporting late-join synchronization, cache hydration, and crash-safe state recovery.

\item Implemented secure WebSocket authentication during the HTTP upgrade handshake with heartbeat-based connection management and deterministic participant identity mapping.

\item Built a production-ready delivery pipeline with Docker, GitHub Actions, Prometheus, Grafana, and an 84-test unit, integration, and end-to-end verification suite.

\end{itemize}
